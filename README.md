#  El-badawy Airlines

A modern dark-themed travel website that showcases popular destinations around the world. This project focuses on clean UI, responsive design, and smooth user experience using pure HTML and CSS.

---

##  Overview

**El-badawy Airlines** is a front-end travel website designed with a night-sky aesthetic. The site presents global destinations categorized by continent, providing users with an engaging and visually appealing browsing experience.

The project demonstrates strong fundamentals in layout structuring, Flexbox usage, responsive design, and modern UI styling.

---

##  Features

*  Categorized destinations by continent
*  Modern dark (night sky) theme
*  Fully responsive design
*  Smooth hover effects
*  Clean and organized layout
*  Beginner-friendly and lightweight

---

##  Built With

* HTML5
* CSS3 (Flexbox & Media Queries)
* Responsive Design Principles
